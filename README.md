Projeto criado com intuito de demonstrar controle financeiro de operações em feiras locais, exemplificando para o feirante as vantagens que um sistema pode trazer para seu empreendimento.

# FeiraMini

Sistema para controle de vendas em feiras livres.

## 🚀 Funcionalidades

- Cadastro de produtos
- Cadastro de feiras
- PDV com resumo de vendas
- Controle de caixa
- Relatórios e gráficos
- Persistência offline (localStorage)

## 📱 Como usar

### Navegador (teste rápido)
1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Comece a usar!
